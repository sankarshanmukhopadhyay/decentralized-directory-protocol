# Glossary

## Namespace
The trust and discovery root under which one or more DeDi registries are published.

## Registry
A collection of records of a defined class under a namespace.

## Record
An individual published item inside a registry.

## Namespace authority
The actor or mechanism that proves control of a namespace.

## Publisher
The actor that creates or updates records.

## Verifier
The relying party that consumes records as part of trust decisions.

## Freshness
The degree to which returned state can be considered current enough for the decision at hand.

## Revocation
A published state indicating that an entity, key, credential, membership, or participant should no longer be treated as valid in a given scope.

## Supersession
A newer state that replaces an older one without necessarily marking the older one as malicious or invalid.
