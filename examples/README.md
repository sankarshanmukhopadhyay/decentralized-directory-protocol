# Example records

These examples are intended to make the repository easier to adopt.

They are illustrative rather than authoritative production data.

## Included examples

- `public-key/` for public key discovery records
- `revoke/` for revocation records
- `membership/` for affiliation and membership records
- `beckn-subscriber/` for network participant records
- `beckn-subscriber-reference/` for registry or record references

## How to use these examples

- compare them with the schemas,
- use them in tests and mocks,
- adapt them into implementation documentation,
- and copy them into consumer integration tutorials.
